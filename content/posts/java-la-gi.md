## Java là gì?
Java là ngôn ngữ lập trình hướng đối tượng,
được sử dụng rộng rãi trong phát triển phần mềm.

## Java trong lập trình mạng
Java cung cấp các thư viện mạnh mẽ như Socket,
ServerSocket hỗ trợ xây dựng ứng dụng client-server.

## Kết luận
Java là nền tảng quan trọng cho sinh viên CNTT.
