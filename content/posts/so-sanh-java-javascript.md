## Điểm giống
- Đều là ngôn ngữ lập trình
- Hỗ trợ lập trình hướng đối tượng

## Điểm khác
- Java chạy trên JVM
- JavaScript chạy trên trình duyệt

## Ứng dụng
Java: backend, mạng  
JavaScript: web, frontend

## Kết luận
Hai ngôn ngữ khác nhau về mục đích sử dụng.
