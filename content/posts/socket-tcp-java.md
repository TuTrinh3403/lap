## TCP là gì?
TCP là giao thức hướng kết nối, đảm bảo dữ liệu chính xác.

## Socket trong Java
Java cung cấp lớp Socket và ServerSocket.

## Ứng dụng
- Chat
- Truyền file
- Client-Server

## Kết luận
TCP phù hợp với các ứng dụng cần độ tin cậy cao.
